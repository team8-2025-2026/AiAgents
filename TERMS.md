# Техническое задание

## Идея проекта

Хотим реализовать чатбота ии-ассистента, помогающий студентам условной обучающей платформы для подготовки к ЕГЭ.

##  Структура нашей абстрактной обучающей платформы:

* Пробники ЕГЭ
* Профильные страницы учителей
  * Преподаватель чего
  * Стаж
  * Краткое описание
* Профильные страницы студентов
  * Предметы, к которым готовится
  * История сдачи пробников

## Функциональные артефакты

### Зачем?

Упростить нагрузку на консультантов и учителей путем передачи типичных задач на руки LLM.

### Для кого?

Студенты обучающей платформы, одиннадцатиклассники -- прямая помощь со стороны чат бота.
Консультанты и учителя -- уменьшение нагрузки.

### Что должен уметь?

- Ответы на вопросы по навигации по сайту
- Помощь с внешними тестами (то есть ответы на вопросы, на которые ответов на обучающей платформе нет)
- Поддержка смены переписки с ии чатботом на переписку с консультантом/учителем
- Сохранение переписки на аккаунте студента, история переписки передается в руки консультанту/учитулю в том и только в том случае, когда с ним ведется беседа

### Критерии приемки

* Реализованный чат для студентов
  * Пользователь может написать свой вопрос в чат бот и ии на него ответит
  * Ответы на типичныые вопросы (вопросы про навигацию по сайту, ...) отвечаются *правильно* и с предоставлением нужных ссылок
  * Пользователь может попросить смену чата к консультанту или конкретному учителю
  * ИИ ассистент умеет предлагать, к кому стоит обратиться в случаях беспомощьности с последующей сменой чата при согласии
* Реализованный чат для консультантов
* Реализованный чат для учителей
