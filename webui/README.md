# WebUI - Интерфейс для AI Agents

Веб-интерфейс для образовательной платформы с поддержкой чатов и управления пользователями.

## Технологии

- React 18
- Vite
- React Router
- CSS3

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Создайте файл `.env` в корне проекта `webui/`:
```
VITE_API_URL=http://localhost:8000
```

3. Запустите dev сервер:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## Структура проекта

```
webui/
├── src/
│   ├── api/          # API клиенты
│   ├── components/   # React компоненты
│   ├── pages/        # Страницы приложения
│   ├── styles/       # CSS стили
│   ├── utils/        # Утилиты
│   ├── App.jsx       # Главный компонент
│   └── main.jsx      # Точка входа
```

## Страницы

- `/auth` - Авторизация
- `/` - Главная страница с чатом (разные интерфейсы для STUDENT/TEACHER/ASSISTENT)
- `/user` - Профиль пользователя
- `/settings` - Настройки (смена пароля)
- `/manage` - Управление пользователями (только для ASSISTENT)

## Функционал

### Студент
- Создание новых чатов
- Переименование чатов
- Удаление чатов
- Отправка и получение сообщений

### Учитель
- Просмотр чатов студентов
- Фильтрация по студенту и статусу
- Переименование чатов
- Отправка и получение сообщений

### Ассистент
- Просмотр всех чатов
- Переименование чатов
- Управление пользователями
- Отправка и получение сообщений

## Интеграция с Auth API

Приложение использует Auth API для:
- Авторизации пользователей
- Получения информации о пользователе
- Управления пользователями
- Смены пароля

API должен быть запущен на `http://localhost:8000` (или указан в `.env`)
